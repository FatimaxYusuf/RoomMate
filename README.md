# ITSE476_project
## Hotel Reservation System

## Overview
This is an open-source Hotel Reservation System that allows users to search for available rooms, make bookings, and manage their reservations.

## Features
- User registration and login  
- Room search and availability  
- Reservation management  

## Development Status
This project is currently in Phase 1, where the basic structure and static pages are being prepared.

## Structure
The project is organized into:
- Frontend  
- Backend  
- Documentation (/docs)
